可进行多选，带折叠收起和展开更多功能，常用于对列表进行筛选。

## API

### TagSelect

| 参数      | 说明                                      | 类型         | 默认值 |
|----------|------------------------------------------|-------------|-------|
| onChange | 标签选择的回调函数 | Function(checkedTags) |  |
| expandable | 是否展示 `展开/收起` 按钮 | Boolean | false |
